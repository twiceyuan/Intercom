<h1 align="center">Intercom</h1>

<p align="center">一个简单的聊天应用，基于 Firebase、Dagger 2、RxAndroid</p>

## 说明

本项目是一个个人模板项目，主要用于整理作者在 Android 开发过程中用到的一些常见的配置，暂时没有实际的需求，希望也可以对有相同目的的朋友提供一个参考，欢迎提出改进的建议。
